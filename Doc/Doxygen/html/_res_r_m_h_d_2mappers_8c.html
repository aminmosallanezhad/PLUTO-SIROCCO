Detailed Description

The PrimToCons() converts an array of primitive quantities to an array of conservative variables for the RRMHD equations.

The ConsToPrim() converts an array of conservative quantities to an array of primitive quantities. During the conversion, primitive quantities are recovered using the algorithm outlined in

  • "Very high order Pn Pm schemes in unstructured meshes for the resistive relativistic MHD equations"
    Dumbser & Zanotti, JCP (2009) 228, 6991-7006. In particular a quartic equation is solved using at first a bracket method and then a Newton-Raphson/Bisection method in order to find the Lorentz factor. Then, assuming ideal EOS, all the primitive quantities are recovered.

Parameters

[out]uprimarray of primitive variables
[in]begstarting index of computation
[in]endfinal index of computation
[in,out]flagarray of flags tagging, in input, zones where entropy must be used to recover pressure and, on output, zones where conversion was not successful.
Returns
Return 0 if conversion was successful in all zones in the range [ibeg,iend]. Return 1 if one or more zones could not be converted correctly and either pressure, density or energy took non-physical values.
