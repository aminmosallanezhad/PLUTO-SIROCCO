Detailed Description

Provides essential functions for allocating and deallocating multi-dimensional arrays.
The functions Array1D(), Array2D(), Array3D(), Array4D() can be used to allocate storage for 1-D, 2-D, 3-D and 4-D arrays of any data type with indices starting at 0. They are typically invoked from within a corresponding macros that handles type casting automatically, e.g.,

char *s;
double **q;
s = ARRAY_1D(20, char);
q = ARRAY_2D(30,40,double);
