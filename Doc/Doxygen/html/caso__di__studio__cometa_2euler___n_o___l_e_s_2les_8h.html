void LES_ViscousFlux ( const Data * d,
Data_Arr rhs,
Grid * grid )

Compute viscous fluxes for the LES model and add them to the rhs according to the sign of

f = N - aux[FMIN]*D,

where N, D > 0. This is executed only if INCLUDE_LES == YES (set in les.h)

aux[FMIN] > 0 filter, put viscous terms where f > 0. aux[FMIN] < 0 do not filter, put viscous terms everywhere.
