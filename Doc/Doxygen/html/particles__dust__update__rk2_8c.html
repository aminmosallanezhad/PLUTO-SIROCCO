Detailed Description


In polar coordinates, particle coordinate and position corresponds to

  • p->coord = (R, phi, z)
  • p->speed = (vR, vphi, vz)

In spherical,

  • p->coord = (r, theta, phi)
  • p->speed = (vr, vtheta, vphi)
