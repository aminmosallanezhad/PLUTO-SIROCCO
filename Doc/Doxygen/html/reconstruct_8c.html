Detailed Description


Common-usgae reconstruction functions (not necessarily optimized) shared by:

  • Finite difference schemes (fd_flux.c);
  • General array reconstructions in the transverse directions (array_reconstruct.c);
  • Some recently added reconstruction schemes (MP5, WENOZ).

These functions returns only the right-extrapolated edge value . In order to obtain also call the same function with array sorted in the reversed direction.

Note that these schemes are not optimized for curvilinear coordinates.
