Detailed Description

The restart structure contains restart information that must be read from disk to restart PLUTO.

Important: The restart structure should be aligned to a power of 2 to prevent (for some compilers) from changing the alignment of the structure and therefore troubleshooting when restarting from files written on different architectures.
