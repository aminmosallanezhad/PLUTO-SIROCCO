Detailed Description

Set plasma (background) initial conditions for standard reference tests for the guiding center approximation (GCA).

The macro SETUP (set in definitions.h) is used to select the configuration:

  • SETUP = 1: simple gyration setup
  • SETUP = 2: ExB drift
  • SETUP = 3: Gradient drift
  • SETUP = 4: Magnetic Null
  • SETUP = 5: E // B test
  • SETUP = 6: Curvature drift
