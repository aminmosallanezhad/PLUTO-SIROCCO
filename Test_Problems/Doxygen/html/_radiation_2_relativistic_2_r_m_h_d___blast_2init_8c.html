Detailed Description


Sets up a 2D radiative magnetized blast wave. Initial conditions are set such that , so that matter dynamics is magnetically dominated when the opacities are low and radiation-dominated otherwise. The latter case also serves to investigate the high-absorption regime in which both the diffusion approximation and LTE are valid.

The radiation field is initially introduced in equilibrium with the gas, with . A null scattering opacity is set for every configuration.

The following setups are implemented:

  • Configuration #01 corresponds to an absorption opacity . In this case the radiation field does not noticeably affect the material's expansion along the -axis, which is magnetically dominated.

  • Configurations #02 and #03 correspond to an absorption opacity , in which gas-radiation interaction produces a much more isotropic acceleration than in Configuration #01. The default IMEX1 method is implemented in Configuration #02 while the IMEX-SSP2(2,2,2) method by Pareschi and Russo (2005) is implemented in Configuration #03. In this problem, IMEX-SSP2(2,2,2) allows for larger time steps than IMEX1.
