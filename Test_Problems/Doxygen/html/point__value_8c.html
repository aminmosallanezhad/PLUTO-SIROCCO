Detailed Description

Reckon variables' punctual values in their respective domains, if high order is enabled, this is done by high order quadrature techniques. High order quadrature uses Laplacian operators

where and are 3D arrays containing, respectively, low order zone-centered and staggered (conservative) variables, and are the Laplacian and transverse laplacian operators. This function performs the convolution of both centered and staggered variables, then converts conservative var. into primitive var.
